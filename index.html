一媒工具

一个自媒体内容管理工具

使用说明

  • 一媒工具是一个自媒体内容管理工具。能将短视频同时发布到抖音、今日头条和西瓜视频。
  • 一媒工具的使用是免费的，但是使用者需要：
    1. 自行申请相关开发账号和权限 （ 抖音,今日头条，西瓜视频, 快手 ）
    2. 完成服务端部署
    3. 安装客户端软件
  • 有任何相关问题，可以发邮件给我。licy2013@aliyun.com

客户端下载和安装

  • 百度网盘

服务端部署

  1. 通过获取服务端代码自行部署
    • github: https://github.com/li-chun-yin/yimei
    • 服务端代码是PHP编写的,可以参考网上的php环境部署方式。
    • 服务端代码使用了 doctrine/orm 操作数据库，理论上这个类库支持的数据库都可以使用。
    • 源代码包含了客户端（client）和服务端（server），服务端部署只需要关心server目录就可以了。
  2. 通过docker镜像，快速部署
    • docker image: https://hub.docker.com/r/lichunyin/yimei
    • Docker 的安装请参考: https://docs.docker.com/get-docker/
    • 安装了docker的环境上，只需要执行 docker run -d -p443:443 lichunyin/yimei 服务端就能运行了
